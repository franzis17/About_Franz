Moving Service Website

What is the website about?

The website provides potential clients an overview of the services offered, and allow them to book or inquire about moving services by using the submission form implemented.

Visit Site

Why make the website?

Client wanted a website as soon as possible for their removalist business. Gathered requirements from the Client such as design ideas, how they want the brand to look, and what functionalities they would need/want.

What did I learn from this project?

  1. UI Design: Based the layout from top 5 removalist companies in Perth with the design being unique as discussed with the client
  2. Valuable Ideas: Desgined the website using Wix and its tools (Gave me ideas about dashboards, messaging systems...etc. for future projects)
  3. Customer Acquisition: Created Google Ad Campaign and performed Search Engine Optimisation (SEO) to gain clients, resulting in first five customers
  4. DNS configuration: Connected custom domain name to website by configuring DNS settings
